# Business Hours

Calculate the number of business hours between two timestamps.

## Getting Started

### Clone the repo

```
git clone https://github.com/heathernfran/business-hours.git
```

### Navigate into the `business-hours` folder

```
cd business-hours
```

### Install dependencies

```
npm install
```

### Run tests

```
npm run test
```

### Run implementations

```
npm start
```
